# Crime AI - 犯罪预测系统

基于公开数据的威胁预测原型

## 灵感来源
THE MACHINE (Person of Interest)

## 核心功能
- 社交媒体情绪分析 → 预测暴力事件
- 网络威胁监控 → 预测攻击
- 异常模式检测 → 识别可疑行为
- 实时威胁分级 → 预测犯罪概率

## 技术栈
- Python + FastAPI
- 情感分析 (VADER/Transformer)
- 关键词监控 (犯罪相关)
- 威胁评分算法
- Telegram 告警

## 数据来源 (公开)
- Twitter/X
- Reddit
- 新闻 API
- 论坛

## 优先级
P0 - 核心预测算法
P1 - 数据采集
P2 - UI 界面
P3 - Telegram 集成
